## 🧰 Common Commands I Use

```bash
# 查看系统负载
top
uptime

# 查看内存
free -h

# 查看磁盘
df -h
du -sh *

# 查看端口
ss -lntup
netstat -lntup

# 查看进程
ps aux
lsof -i

# 查看服务状态

tail -f /var/log/messages
```
